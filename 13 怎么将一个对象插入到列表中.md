## 怎么将一个对象插入到列表中

解法1：用 insert 方法


```
>>> num = [1, 2, 4, 5]
>>> num.insert(2, 'three')
>>> num
[1, 2, 'three', 4, 5]
```
解法2：用切片的方式插入

```
>>> num = [1, 2, 4, 5]
>>> num[2:2] = ['three']
>>> num
[1, 2, 'three', 4, 5]
```
