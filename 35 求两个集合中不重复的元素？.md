## 求两个集合中不重复的元素？

差集指的是两个集合交集外的部分

解法1: 使用运算符 ^


```
>>> a = {6, 7, 8}
>>> b = {7, 8, 9}
>>> a ^ b
{9, 6}
```

解法2:使用 symmetric_difference 方法


```
>>> a = {6, 7, 8}
>>> b = {7, 8, 9}
>>> a.symmetric_difference(b)
{9, 6}
```
