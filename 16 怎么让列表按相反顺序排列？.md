## 怎么让列表按相反顺序排列？

解法1：用 reverse 方法

```
>>> num = [1, 22, 45, 99, 49]
>>> num.reverse()
>>> num
[49, 99, 45, 22, 1]
```
解法2：用切片的方式

```
>>> num = [1, 22, 45, 99, 49]
>>> num[::-1]
[49, 99, 45, 22, 1]
```
