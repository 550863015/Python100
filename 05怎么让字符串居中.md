## 怎么让字符串居中

用字符串中的 center 方法，他会在两边自动填充字符（默认为空格），让字符串居中

```
>>> k = '更多精彩，请关注公众号「Python知识圈」'
>>> k.center(50)
'              更多精彩，请关注公众号「Python知识圈」              '
>>> k.center(50, '*')
'**************更多精彩，请关注公众号「Python知识圈」**************'
```
