## 求m中元素在n中出现的次数

基础解法：


```
>>> m = {'A', 'B', 'C'}
>>> n = {'B', 'C', 'D'}
>>> found = 0
>>> for i in m:
    if i in n:
        found += 1
        
>>> found
2
```
高级解法：

```
>>> m = {'A', 'B', 'C'}
>>> n = {'B', 'C', 'D'}
>>> len(m & n)
2
```

如果m和n不是集合的话，直接转换后再取交集

```
>>> m = {'A', 'B', 'C'}
>>> n = {'B', 'C', 'D'}
>>> len(set(m) & srt(n))
2
```

高级解法的另一种写法：


```
>>> m = {'A', 'B', 'C'}
>>> n = {'B', 'C', 'D'}
>>> len(set(m).intersection(n))
2
```
